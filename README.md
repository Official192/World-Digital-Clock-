📖 About
The World Digital Clock is a lightweight, responsive web application designed to provide accurate timekeeping with a modern aesthetic. Built using vanilla JavaScript, it features a stunning "frosted glass" UI (Glassmorphism) that floats over a dynamic gradient background. It requires no external frameworks or libraries—just pure code.

Live Demo: Click Here to View

✨ Features
🕐 Real-Time Updates: Refreshes every second (1000ms) for accuracy.
🎨 Glassmorphism UI: Modern frosted glass design with backdrop-filter blur effects.
🌍 Full Date Support: Displays the Day, Month (Full Name), and Year.
📱 Fully Responsive: Looks perfect on desktops, tablets, and mobile devices.
🚀 Zero Dependencies: Single HTML file solution.
